# Block UI

## Show
> Method Name : showBlockUI(message = "Please wait a moment ...")

### Parameter
- message (string)

### How to call
- basic
```javascript
showBlockUI()
```
- with custom message
```javascript
showBlockUI('loading...')
```

## Hide
> Method name : hideBlockUI()

### How to call
- basic
```javascript
hideBlockUI()
```
